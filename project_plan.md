# VisaBot — Панель управления ботом для визового центра Испании

## 1. Описание проекта
Веб-панель управления для автоматизированного бота записи в визовый центр Испании в России. Панель позволяет управлять клиентами, очередью, серверами, видео-верификацией и мониторить работу бота 24/7.

**Целевая аудитория:** Операторы и супер-админы визового агентства
**Ключевая ценность:** Централизованное управление всеми процессами автоматической записи

## 2. Структура страниц
- `/login` — Страница входа (роли: супер-админ, оператор)
- `/dashboard` — Главная панель управления (статистика, статус бота, быстрые действия)
- `/clients` — База клиентов (добавление, редактирование, поиск, фильтры)
- `/queue` — Очередь клиентов (приоритеты, пауза, удаление)
- `/slots` — Мониторинг слотов (статус поиска, история найденных слотов)
- `/servers` — Конфигурация серверов (российские IP, переключение)
- `/videos` — Видео-верификация (загрузка, просмотр, привязка к клиентам)
- `/logs` — Логи работы и история действий
- `/settings` — Настройки системы (скорость проверки, уведомления)

## 5. План интеграций
- **Supabase**: Будет подключен позже для реальной базы данных, авторизации и хранения видео
- **Backend bot**: Отдельный серверный бот будет подключаться к панели через API (будущая интеграция)

## 6. План разработки по фазам

### Фаза 1: Авторизация и навигация
- **Цель**: Создать систему входа с ролями и основной layout панели
- **Результат**: Страница логина, sidebar-навигация, защищённые маршруты

### Фаза 2: Дашборд и статистика
- **Цель**: Создать главную страницу панели с ключевыми метриками
- **Результат**: Виджеты статистики, статус бота, последние события, графики

### Фаза 3: База клиентов
- **Цель**: Полноценное управление клиентами
- **Результат**: Таблица клиентов, формы добавления/редактирования, фильтры, поиск

### Фаза 4: Очередь и слоты
- **Цель**: Управление очередью и мониторинг слотов
- **Результат**: Очередь с drag-and-drop, статус поиска слотов, история

### Фаза 5: Серверы и видео
- **Цель**: Управление серверами и видео-верификацией
- **Результат**: Список серверов, загрузка видео, привязка к клиентам

### Фаза 6: Логи и настройки
- **Цель**: Система логов и конфигурация
- **Результат**: Логи с фильтрами, настройки скорости и уведомлений
